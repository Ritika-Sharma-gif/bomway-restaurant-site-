# Bombay Family Restaurant & Bar

This project is ready for free hosting on Netlify or GitHub Pages.

## Deploy on Netlify
1. Go to https://app.netlify.com/drop
2. Drag and drop the project folder (the folder containing this README and the live folder)
3. Netlify will publish a public URL immediately.

## Deploy on GitHub Pages
1. Upload this project to a GitHub repository.
2. Open Settings > Pages.
3. Select the main branch and the root folder.
4. GitHub will generate a public URL.

The root redirect page will send visitors to the main site in the live folder.
